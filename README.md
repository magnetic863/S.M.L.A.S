# S.M.L.A.S. (Static Magnetic Lattice Array System)

**English Summary:**
This project presents a theoretical model for a passive magnetic levitation platform using hexagonal matrix arrays. It aims to bypass Earnshaw's theorem through lattice smoothing, lever stabilization, and eddy current damping.

---

### Концепция пассивной магнитной левитации на базе гексагональных матриц

## 1. Аннотация (Abstract)
Проект представляет собой теоретическую модель крупногабаритной транспортной или выставочной платформы, использующей **полностью пассивную магнитную левитацию**. В отличие от классических систем, данная система опирается на геометрическое распределение магнитных потоков и механическую жесткость конструкции.

## 2. Физические принципы
Система использует принцип **дискретной силовой среды**. Вместо создания единого выпуклого магнитного поля, которое неизбежно привело бы к нестабильности по теореме Ирншоу, пол разделен на тысячи независимых гексагональных ячеек.

**Ключевые решения:**
* **Магнитные "фонтанчики":**
![Магнитная матрица](1770297648025.png)
  Изолированные столбики создают направленные вверх векторы силы (S-полюс), предотвращая слияние полей в общую "горку".
* **Стальные волноводы:** Каждый шестигранник закован в трансформаторную сталь 3408 по бокам. Это фокусирует магнитный поток строго вертикально, исключая боковое взаимодействие с соседями.
* **Распределитель нагрузки:** На левитирующих дисках установлена тончайшая сталь с "козырьками", которая размазывает точечные удары магнитных иголок и защищает от опрокидывания.


## 3. Ключевые узлы системы
### А. Магнитная база (Stator)
* **Геометрия:** Гексагональная упаковка (Hexagonal Close Packing).
* **Элементы:** Неодимовые цилиндры марки **SH/UH**.

### Б. Платформа (Levitator)
* **Опоры:** 6 магнитных дисков с осевой намагниченностью.
* **Flux Refinement:** Ультратонкая (0.1 мм) трансформаторная сталь.
* **Каркас:** Пространственная ферма из композита или алюминия.

## 4. Технические преимущества
1. **Zero-Power:** Система не потребляет электроэнергию.
2. **Долговечность:** Минимальная деградация поля NdFeB.
3. **Стабильность:** Шеститочечная схема исключает эффект «качающегося стола».

## 5. Материалы (BOM)
* Магниты: **NdFeB N52SH**.
* Стабилизаторы: Листовая медь (>99%).
* Концентраторы: Сталь 3408.
